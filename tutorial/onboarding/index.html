Getting Started with Connectlift Mate 💙

Simple walkthrough to help you connect smarter, faster and stress-free on LinkedIn.

🧩 Step 1
Install the Extension
Find Connectlift Mate on the Chrome Web Store → Click Install → Pin it to your toolbar for easy access.
📝 Step 2
Add Your First Note
Click the extension icon → Add a Note → Type your message → Save it. Your note is now ready for LinkedIn use.
⭐ Step 3
Set a Default Note
Make one note your primary note by selecting “Set as Default” — it’ll always appear first.
📩 Step 4
Insert Notes on LinkedIn
Click “Connect” on a LinkedIn profile → Open the extension → Hit “Insert This” → Done!
💬 Step 5
Use Message Templates
Switch to the Messages tab → Save replies, follow-ups, outreach templates → Copy or insert in one click.
🔍 Step 6
Build Boolean Search Queries
Use the Boolean Builder tab → Generate smart LinkedIn search strings → Paste into the LinkedIn search box.
⚙ Step 7
Explore Settings
Export/import your data, clear saved notes/messages, toggle themes, and personalize your experience.
🚀 Step 8
More Features Coming Soon
Get ready for Drag & Drop reordering, Dark Mode, Message Analyzer, Template Pinning and more — all in upcoming updates.
